# AI customer tasks
Forestry AI customer service client to train RAG models e.g. categorization of SalesForce cases, generative models e.g. SalesForce answers, and model compression e.g. executable on phones.

## Getting started

### Prerequisites

- Python 3.7 or later is required to use this package.
- Optional SalesForces subscription

### Install the package

Install the Azure ML client library for Python with pip

```bash
pip install azure-ai-ml
pip install azure-identity
```
